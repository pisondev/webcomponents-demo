# Penjelasan Pengembangan

Berdasarkan dari Hands-On Modul yang telah dibuat, dilakukan pengembangan modifikasi sebagai berikut:

## Components
- **my-info.js:** nama + nim.
- **header.js:** header (mata kuliah).
- **layout-two-column.js:** susunan tata letak tampilan web-nya.
- **product-card.js:** nama produk + harga.
- **cart-summary.js:** rangkuman nota struk pembayarannya.
- **counter-display & counter-controls (modifikasi):** versi sedikit lebih rapi.

---

File index.html sebagai tempat tampilan web utama yang terdiri dari file komponen-komponen sebelumnya yang telah disebutkan.
